# SpringBoard

SpringBoot+JPA를 공부하고 만든 게시판 프로젝트

## 1. 프로젝트 기능

* 회원가입/로그인 기능
* 게시글 작성/조회/수정/삭제 기능
* 게시글에 댓글 작성 기능
* 제목/카테고리별 검색 기능
* 게시글 조회수 표시
* 페이징 처리


### 3. 개발환경

*백엔드
  JAVA 17
  SpringBoot 3.2.1
  JPA(Spring Data JPA)
  Spring Security

*프론트엔드
  html/css
  thymeleaf
  bootstrap

*database
  MariaDB
